Voice Dictation Guide
+
Say each item on one line, fields separated by commas:
[type spec], [location path], [qty], [condition], [frequency], [criticality], [replaceability], [keep], [year]
"Batteries AAA, Organizer right angle bracket Shelf 2, two packs, new, monthly, nice to have, easy cheap, keep, 2024"
LOCATION: Say "right angle bracket" between path levels
QTY: one / two / three / a pack / two packs / a pair / a few
COND: new / good / fair / poor
FREQ: daily / weekly / monthly / rarely / never
CRIT: essential / important / nice to have / redundant
REPL: irreplaceable / hard expensive / moderate / easy cheap
KEEP: keep / maybe / sell / donate / trash
NAME: type first then spec then brand
S/N: add "S/N" then number at end

Location System
Path hierarchy
Locations are path arrays: ["Organizer","Shelf 2","Box A"]. No depth limit. Displayed as Organizer > Shelf 2 > Box A.

Click expand arrow to open a node. Click name to filter. Counts include all descendants.
Manage Locations
Header button: Rename, Move Under, Wrap in Parent. All item paths update automatically.
Editing
Click-to-edit
Click any cell. Tab/Shift+Tab moves fields. Shortcut keys shown below open dropdowns.

Bulk ops
Check rows → orange bar. Bulk-set location, category, doc type, digital copy, scoring fields.
